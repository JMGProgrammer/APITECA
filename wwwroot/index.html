APITECA (DEMO)

Mi primer API Web funciona. Funciona como ejemplo de gestión de una biblioteca personal, fue desarrollada con ASP.NET Core 10 y Entity Framework Core usando MySQL como motor de base de datos.

Permite administrar una colección de libros: agregar, modificar, eliminar, buscar, y obtener estadísticas y recomendaciones basadas en la cantidad de páginas de cada libro.

Modelo de Datos - Libro

CampoTipoDescripción
IdintIdentificador único (autoincremental)
TitulostringTítulo del libro
AutorstringNombre del autor
PaginasintCantidad de páginas
LeidobooleanIndica si el libro ha sido leído

Endpoints

Base URL: /api/libros

GET /api/libros

Retorna la lista completa de libros almacenados en la base de datos.

[
  {
    "id": 1,
    "titulo": "El Hobbit",
    "autor": "J.R.R. Tolkien",
    "paginas": 310,
    "leido": true
  }
]

GET /api/libros/{id}

Busca y retorna un libro específico por su ID.

Parámetros: id (int) - Identificador del libro.

{
  "id": 1,
  "titulo": "El Hobbit",
  "autor": "J.R.R. Tolkien",
  "paginas": 310,
  "leido": true
}

404:

"Libro no encontrado"

GET /api/libros/{id}/tipo

Clasifica un libro según su cantidad de páginas.

Rango de páginasClasificación
Menos de 100Libro Corto
100 - 300Libro Mediano
Más de 300Libro Largo
 
{ "tipo": "Libro Mediano" }

GET /api/libros/{id}/fibonacci

Calcula el número de Fibonacci correspondiente al número de páginas del libro.

{
  "titulo": "El Hobbit",
  "paginas": 310,
  "numero": 310,
  "fibonacci": 167769361818756559872424281766416725811394149890590732499100663797129936
}

GET /api/libros/autor/{autor}

Retorna todos los libros escritos por el autor indicado (no distingue mayúsculas/minúsculas).

[
  {
    "id": 1,
    "titulo": "El Hobbit",
    "autor": "J.R.R. Tolkien",
    "paginas": 310,
    "leido": true
  }
]

404:

"No se encontraron libros para el autor especificado"

GET /api/libros/resumen

Retorna estadísticas generales de la biblioteca.

{
  "totalLibros": 10,
  "librosLeidos": 6,
  "librosNoLeidos": 4,
  "promedioPaginas": 245.5
}

POST /api/libros

Agrega un nuevo libro a la base de datos. El campo Id se genera automáticamente.

Body (application/json):

{
  "titulo": "El Hobbit",
  "autor": "J.R.R. Tolkien",
  "paginas": 310,
  "leido": true
}

200:

{
  "id": 1,
  "titulo": "El Hobbit",
  "autor": "J.R.R. Tolkien",
  "paginas": 310,
  "leido": true
}

400:

"Libro no válido"

POST /api/libros/con-recomendacion

Agrega un libro y retorna una recomendación de lectura según la cantidad de páginas.

PáginasRecomendación
< 100Ideal para leer en un día
100 - 300Lectura normal
> 300Lectura larga

Body (application/json):

{
  "titulo": "El Hobbit",
  "autor": "J.R.R. Tolkien",
  "paginas": 310,
  "leido": true
}

200:

{
  "libro": {
    "id": 1,
    "titulo": "El Hobbit",
    "autor": "J.R.R. Tolkien",
    "paginas": 310,
    "leido": true
  },
  "recomendacion": "Lectura larga"
}

PUT /api/libros/{id}

Actualiza todos los campos de un libro existente.

Parámetros: id (int) - Identificador del libro a modificar.

Body (application/json):

{
  "titulo": "El Hobbit (Edición Actualizada)",
  "autor": "J.R.R. Tolkien",
  "paginas": 320,
  "leido": true
}

404:

"Libro no encontrado"

PUT /api/libros/{id}/marcar-leido

Marca el campo Leido del libro como true. No requiere body.

200:

"El libro ha sido marcado como leído."

404:

"Libro no encontrado"

DELETE /api/libros/{id}

Elimina permanentemente un libro de la base de datos.

200:

{
  "id": 1,
  "titulo": "El Hobbit",
  "autor": "J.R.R. Tolkien",
  "paginas": 310,
  "leido": true
}

404:

"Libro no encontrado"
